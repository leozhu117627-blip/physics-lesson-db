⚙️ 連線設定

設定 GitHub 資訊以連接雲端教案資料庫

GitHub 設定

教案資料庫存放在您的 GitHub Repository，需要以下資訊才能讀取和更新。

在 GitHub → Settings → Personal access tokens 建立，勾選 repo 權限即可。Token 僅存於您的瀏覽器，不會傳送至任何伺服器。

部署說明

首次使用請依照 SETUP.md 完成以下步驟：

  1. 建立 Public GitHub Repository 並上傳 index.html 和 db/lessons.json
  2. Repository → Settings → Pages → 開啟 GitHub Pages（main / root）
  3. 建立 PAT（只需勾選 repo）→ 填入上方設定
